题目描述：

给定三个串 a,b,sa,b,sa,b,s，其中 a,ba,ba,b 长度为 nnn，sss 长度为 mmm，求出四元组 (l1,r1,l2,r2)(l1,r1,l2,r2)(l1,r1,l2,r2) 的个数，满足：

  1. [l1,r1][l1,r1][l1,r1] 和 [l2,r2][l2,r2][l2,r2] 的交集非空。

  2. aaa 中位置 l1l1l1 到 r1r1r1 组成的子串与 bbb 中位置 l2l2l2 到 r2r2r2 组成的子串拼起来恰好是 sss。

1≤n≤5000001 \leq n \leq 5000001≤n≤500000，2≤m≤2n2 \leq m \leq 2n2≤m≤2n。

题目分析：

如果没有第一个条件，直接 AC 自动机做就好了。

有了第一个条件之后，AC 自动机显然做不了。

考虑一个串肯定是一个前缀和一个后缀拼起来的，那么我们用 lcpailcpa_ilcpai​ 表示 ai,ai+1,⋯ ,ana_i,a_{i+1},\cdots,a_nai​,ai+1​,⋯,an​ 和 s1,s2,⋯ ,sn−1s_1,s_2,\cdots,s_n-1s1​,s2​,⋯,sn​−1 的最长公共前缀，lcpbilcpb_ilcpbi​ 表示 b1,b2,⋯ ,bib_1,b_2,\cdots,b_ib1​,b2​,⋯,bi​ 和 s2,s3,⋯ ,sns_2,s_3,\cdots,s_ns2​,s3​,⋯,sn​ 的最长公共前缀。

以上用 Z函数 求出即可。

需要保证：

r1−l1+1+r2−l2+1=mr_1-l_1+1+r_2-l_2+1=m r1​−l1​+1+r2​−l2​+1=m

移一下项，得到：

r2=m−r1+l2+l1−2r2=m-r_1+l_2+l_1-2 r2=m−r1​+l2​+l1​−2

因为两区间有交，那么肯定有 r1≤l2∧l1≤r2r_1\le l_2 \land l_1\le r_2r1​≤l2​∧l1​≤r2​，所以 −r1+l2≤0-r_1+l_2\le0−r1​+l2​≤0，即：

l1≤r2≤m+l1−2l_1 \le r_2\le m+l_1-2 l1​≤r2​≤m+l1​−2

r1r_1r1​ 和 l2l_2l2​ 就不需要管了，只要选择的两个点符合如上条件，便符合题意。

考虑得到 l1l_1l1​ 和 r2r_2r2​ 之后如何求四元组个数，其实就是 lcpal1+lcpbr2−m+1lcpa_{l_1}+lcpb_{r_2}-m+1lcpal1​​+lcpbr2​​−m+1，如下图：

那么红色区间中的任何一个点都可以被作为分界点，能组成的四元组个数自然也就是分界点的个数。

那么统计答案从五维变成两维：

∑l1=1n∑r2=l1min⁡(n,l1+m−2)max⁡(lcpal1+lcpbr2−m+1,0)\sum_{l_1=1}^{n} \sum_{r_2=l_1}^{\min(n,l_1+m-2)} \max(lcpa_{l_1}+lcpb_{r_2}-m+1,0) l1​=1∑n​r2​=l1​∑min(n,l1​+m−2)​max(lcpal1​​+lcpbr2​​−m+1,0)

肯定需要降到 log⁡\loglog 层次，但是这个 max⁡\maxmax 搞得很难受，也就是说 lcpal1+lcpbr2−m+1≥0lcpa_{l_1}+lcpb_{r_2}-m+1\ge 0lcpal1​​+lcpbr2​​−m+1≥0，即 lcpal1≥m−lcpbr2−1lcpa_{l_1}\ge m-lcpb_{r_2}-1lcpal1​​≥m−lcpbr2​​−1。

式子变为：

∑l1=1ncnt×(lcpal1−m+1)∑r2=l1min⁡(n,l1+m−2)lcpbr2[m−lcpbr2−1≤lcpal1]\sum_{l_1=1}^{n} cnt\times (lcpa_{l_1}-m+1) \sum_{r_2=l_1}^{\min(n,l_1+m-2)} lcpb_{r_2}[ m-lcpb_{r_2}-1\le lcpa_{l_1}] l1​=1∑n​cnt×(lcpal1​​−m+1)r2​=l1​∑min(n,l1​+m−2)​lcpbr2​​[m−lcpbr2​​−1≤lcpal1​​]

发现可以用两个树状数组来求值，一个是当前合法区间内 lcpbr2lcpb_{r_2}lcpbr2​​ 出现过得次数，一个是当前合法区间内 lcpbr2lcpb_{r_2}lcpbr2​​ 出现的和。
