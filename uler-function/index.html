欧拉筛的推导，和一些欧拉函数的性质

欧拉函数的几个性质（以下 ppp 无特殊说明，都为质数）：

  1. 若 gcd(a,b)=1gcd(a,b)=1gcd(a,b)=1 ，则 φ(a)×φ(b)\varphi(a)\times \varphi(b)φ(a)×φ(b)，证明不会，大家记住就行了

  2. n=∑d∣nφ(d)n=\sum_{d|n}\varphi(d)n=∑d∣n​φ(d)

    证明：设 f(x)f(x)f(x) 表示 gcd(k,n)=xgcd(k,n)=xgcd(k,n)=x 的数的个数（ k<=nk<=nk<=n ）

    结论：n=∑i=1nf(i)n=\sum_{i=1}^nf(i)n=∑i=1n​f(i)

    我们可以分类讨论（保证 x<=nx<=nx<=n）：

    1. xxx 是 nnn 的约数，那么 gcd(x,n)=xgcd(x,n)=xgcd(x,n)=x ，可以有 111 的贡献
    2. xxx 不是 nnn 的约数，xxx 也和 nnn 不互质，那么一定会被他们俩的最大公约数带来 111 的贡献
    3. xxx 和 nnn 互质，那么会被 gcd(x,n)=1gcd(x,n)=1gcd(x,n)=1 带来 111 的贡献

    综上：每个数 xxx 都会且只会被统计一次 111 的贡献

    证出结论：n=∑i=1nf(i)n=\sum_{i=1}^nf(i)n=∑i=1n​f(i)

    现在我们观察上面的式子：gcd(k,n)=dgcd(k,n)=dgcd(k,n)=d 等价于 gcd(kd,nd)=1gcd(\frac{k}{d},\frac{n}{d})=1gcd(dk​,dn​)=1 也就等价于 φ(nd)\varphi(\frac{n}{d})φ(dn​)

    所以上面的结论可以转化成：n=∑d∣nφ(nd)=∑d∣nφ(d)n=\sum_{d|n}\varphi(\frac{n}{d})=\sum_{d|n}\varphi(d)n=∑d∣n​φ(dn​)=∑d∣n​φ(d)

  3. 若 n=pkn=p^kn=pk ，则 φ(n)=pk−pk−1\varphi(n)=p^k-p^{k-1}φ(n)=pk−pk−1

    可以看出用到了容斥，用总的数量，减去和 nnn 不互质的数量。

    法一：设 xxx 为 ppp 的系数，则 xxx 的取值范围为 1,2,3,...,pk−11,2,3,...,p^{k-1}1,2,3,...,pk−1 ，所以总共有 pk−1p^{k-1}pk−1 个与 nnn 不互质的数

    法二：pk/pp^k/ppk/p 就是 ppp 的倍数在 1−n1-n1−n 中出现的个数，即 pk−1p^{k-1}pk−1

  4. φ(pk)=pk−1×(p−1)\varphi(p^k)=p^{k-1}\times(p-1)φ(pk)=pk−1×(p−1)

    这个证明是简单的

    上面已经得出 φ(pk)=pk−pk−1\varphi(p^k)=p^k-p^{k-1}φ(pk)=pk−pk−1 我们把 pk−1p^{k-1}pk−1 提出来

    可以得到 φ(pk)=pk−1×(p−1)\varphi(p^k)=p^{k-1}\times(p-1)φ(pk)=pk−1×(p−1)

  5. φ(n)=n∗∏i=1cpi−1pi\varphi(n)=n*\prod_{i=1}^c\frac{p_i-1}{p_i}φ(n)=n∗∏i=1c​pi​pi​−1​ （ ccc 是 nnn 的质因数个数）

    证明：

    已知 n=∏i=1cφ(piki)n=\prod_{i=1}^c \varphi(p_i^{k_i})n=∏i=1c​φ(piki​​) （证明：由 111 性质可知）

    则 n=∏i=1c(pk−pk−1)n=\prod_{i=1}^c (p^k-p^{k-1})n=∏i=1c​(pk−pk−1) 这次我们提出来 pkp^kpk

    式子就变成 n=∏i=1c(pk×(1−1p))n=\prod_{i=1}^c(p^k\times(1-\frac{1}{p}))n=∏i=1c​(pk×(1−p1​))

了解了这些性质以后，我们就可以愉快的进行欧拉筛了

分类讨论：

  1. 若 ppp 为素数，则 φ(p)=p−1\varphi(p)=p-1φ(p)=p−1

  2. 发现 nnn 会被 p×n′p\times n'p×n′ 筛掉

    1. ppp 是 n′n'n′ 的最小质因数，φ(n)=p×φ(n′)\varphi(n)=p\times \varphi(n')φ(n)=p×φ(n′)

      证明：

      φ(n)=n×∏i=1cpi−1pi\varphi(n)=n\times\prod_{i=1}^c\frac{p_i-1}{p_i}φ(n)=n×∏i=1c​pi​pi​−1​

      把 nnn 分解成 p×n′p\times n'p×n′ 可知 φ(n)=p×n′×∏i=1cpi−1pi\varphi(n)=p\times n'\times\prod_{i=1}^c\frac{p_i-1}{p_i}φ(n)=p×n′×∏i=1c​pi​pi​−1​

      把后两项合并得： φ(n)=p×φ(n′)\varphi(n)=p\times \varphi(n')φ(n)=p×φ(n′)

    2. ppp 不是 n′n'n′ 的最小质因数

      则 φ(n)=φ(n′)×p\varphi(n)=\varphi(n')\times pφ(n)=φ(n′)×p

      即 φ(n)=φ(n′)×(p−1)\varphi(n)=\varphi(n')\times(p-1)φ(n)=φ(n′)×(p−1)
